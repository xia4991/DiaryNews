## Phase 1: Backend Auth

1. **Dependencies** вЂ” Add `PyJWT`, `google-auth`, `google-auth-oauthlib` to `requirements.txt`; pin all packages
2. **Config** вЂ” Add `JWT_SECRET`, `GOOGLE_CLIENT_ID`, `GOOGLE_CLIENT_SECRET`, `ADMIN_EMAILS`, `CORS_ORIGINS`, `DB_PATH` to `backend/config.py` from env
3. **Users table** вЂ” Add to `backend/database.py`: users table with google_id, email, name, avatar, is_admin; enable WAL mode, timeout=10
4. **User storage** вЂ” New `backend/storage/users.py`: `get_or_create_user()`, `get_user_by_id()`
5. **Auth module** вЂ” New `backend/auth.py`: Google token verification, JWT create/decode, FastAPI dependencies (`get_current_user`, `require_admin`)
6. **Auth endpoints** вЂ” `POST /api/auth/google` (login), `GET /api/auth/me` (current user)
7. **Protect endpoints** вЂ” Public: news/status. Login required: YouTube/Ideas. Admin only: fetch/manage

## Phase 2: Frontend Auth

1. **Install** вЂ” `npm install @react-oauth/google`
2. **API client** вЂ” Add interceptors (attach JWT, handle 401), auth methods, timeout
3. **Auth context** вЂ” New `react-frontend/src/auth.jsx`: GoogleOAuthProvider + user state + useAuth hook
4. **Entry point** вЂ” Wrap App with AuthProvider in `main.jsx`
5. **Login page** вЂ” New `react-frontend/src/pages/LoginPage.jsx`: dark theme + Google sign-in button
6. **App.jsx** вЂ” Conditional tabs (news always, YouTube/Ideas when logged in), login flow
7. **Header** вЂ” Login/logout buttons, user avatar, admin-only fetch buttons
