# NIF、NISS 与 SNS 号码基础说明

## Summary

NIF、NISS 和 SNS utente number 是在葡生活中最常见的三个基础号码。它们分别对应税务、社保和公立医疗系统，不能互相替代。

## What It Means

NIF 常用于买卖、签合同、开户和税务处理。NISS 关系到社保权利义务，也常和工作关系绑定。SNS utente number 用于进入公立医疗系统，但“有 utente number”并不自动等于所有医疗费用都由 SNS 承担。

## Key Steps / Rules

- 先分清三个号码的机构和用途：NIF 对应税务，NISS 对应 Segurança Social，utente number 对应 SNS
- gov.pt 说明，NIF 是做买卖、签合同、开户等很多事务的基础号码
- gov.pt 说明，NISS 目前可在线申请，外国公民、某些葡籍人士、法定代表人和雇主都可能是申请主体
- gov.pt 说明，外国人首次去公立医疗单位时通常就可以获得 SNS utente number，也可以主动到公立医疗单位申请
- 对于 SNS 覆盖医疗费用，gov.pt 特别说明，仅有 utente number 本身并不保证费用一定由 SNS 承担；还可能需要把身份证件、葡萄牙 NIF、葡萄牙住址和有效居留关联到系统
- 保存好号码证明和截图，后续开户、签工、报税、就医时都会反复用到
- 2026 年 gov.pt 还提供了一个在部分 Espaço Cidadão 一次性申请 NIF、NISS 和 SNS 号码的服务，但适用条件和办理点有限，需要看当期官方页面

## Important Caveats

- 不同身份状态和住址状态，办理材料可能不同
- 对非居民 NIF，是否需要 fiscal representative，要看是否居住在欧盟/欧洲经济区以外等情形
- NISS 和 SNS 编号都不等于自动取得全部对应权利，很多情况下还要完成额外关联或资格核验
- 如果柜台说法和旧攻略不一致，优先核对当前 gov.pt 页面
