This directory is the only knowledge source for AI Assistant version `1.0`.

Rules for this wiki:

- every answer must be grounded in markdown content from this folder
- keep one topic per page when possible
- prefer clear headings over long unstructured text
- update the `Last reviewed` field when content changes

Recommended structure:

- `immigration/`
- `law/`
- `living/`
- `work/`
- `services/`
